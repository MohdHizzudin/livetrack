# 📍 LiveTrack

**Sistem Live Location Sharing** web-based yang cantik, selamat & mudah digunakan.

### Fitur Lengkap
- Login Google SSO (1 klik)
- Share live location (toggle on/off)
- Peta real-time dengan multiple user
- Buat & manage Group
- Admin Dashboard (approve user, tambah manual, analytics)
- SOS Button + alert
- History perjalanan 24 jam
- PWA (boleh install di phone macam app)
- Dark/Light mode + responsive mobile

### Teknologi
- Next.js 15 (App Router)
- Firebase (Auth + Realtime Database)
- Google Maps API
- Tailwind + Lucide Icons

### Cara Setup
1. `npm install`
2. Buat project Firebase (enable Google Auth & Realtime Database)
3. Isi `.env.local` (lihat `.env.example`)
4. Dapatkan Google Maps API Key
5. `npm run dev`

### Deploy
- **Vercel** (paling mudah)
- Firebase Hosting
